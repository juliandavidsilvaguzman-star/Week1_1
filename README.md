# Week1_1 - Deep Learning

Entiende una neurona en 15 minutos
Lógica matemática + Python + NumPy + Machine Learning básico

## Descripción general
Este repositorio contiene una guía de nivelación práctica cuyo objetivo es proporcionar las bases mínimas necesarias para iniciar un curso de Deep Learning con confianza.
El énfasis no está en la teoría avanzada, sino en comprender los conceptos clave, ejecutar notebooks correctamente, interpretar resultados y ajustar modelos de forma consciente.

## Objetivos de aprendizaje
Al finalizar esta guía, el estudiante será capaz de:

Comprender la lógica matemática detrás de una decisión binaria.
Implementar funciones de decisión usando Python.
Usar NumPy para operar con vectores y matrices.
Entender el flujo básico de Machine Learning: entrenar, evaluar y ajustar.
Ejecutar y modificar notebooks en Google Colab.
Preparar evidencia académica/profesional para GitHub.


## Contenidos desarrollados

- Proposiciones (verdadero / falso)
- Operadores lógicos: AND, OR, NOT
- Condicionales (if / else)
- Umbrales y su impacto en la clasificación
- Función de decisión:
    # z=x⋅w+bz = x \cdot w + bz=x⋅w+b

- Clasificación binaria (Clase 0 / Clase 1)
- Diferencia entre función Step y Sigmoid


# 2. Python esencial

- Variables numéricas y listas
- Definición de funciones (def)
- Estructuras condicionales
- Ciclos for
- Interpretación de errores comunes

# 3. NumPy y Machine Learning básico

- Creación de arreglos, vectores y matrices
- Producto matricial (@)
- Cálculo del puntaje z
- Implementación de un perceptrón simple (compuerta AND)
- Simulación de un mini entrenamiento con ajuste de pesos y bias
- Introducción a conceptos como:
    Entrenamiento
